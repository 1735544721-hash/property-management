摘  要

在小区物业管理中，住户报修、费用缴纳、公告通知与投诉处理等业务分散在不同系统或线下流程中，信息孤岛与状态不可追溯导致工单响应迟缓、费用核算困难和服务评价不透明；同时住户、物业人员与管理者的多角色协同缺乏统一权限与数据闭环，影响社区治理与服务效率。本文面向小区物业管理业务，设计并实现前后端分离的信息管理系统，前端采用 Vue 3 与 Element Plus 构建住户端与管理端界面，后端基于 Spring Boot 3.2、MyBatis-Plus、Spring Security、JWT 与 MySQL 搭建服务层，覆盖用户与权限、楼栋房屋、报修工单、费用管理、公告发布、投诉处理与数据看板等模块，通过流程流转、提醒机制与统计分析提升服务响应与费用可追溯性，为社区治理与运营决策提供可靠数据支撑。

关键词：小区物业管理；报修工单；费用管理；Spring Boot
