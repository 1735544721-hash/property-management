# 技术栈与依赖

## 后端
- Java 17（`property-management/小区物业管理系统/property-management-backend/pom.xml`）
- Spring Boot 3.2.1 / Spring Web / Spring Security / Validation
- MyBatis-Plus 3.5.5
- MySQL Connector/J
- Druid 1.2.20
- JWT（`jjwt` 0.12.3）
- Hutool 5.8.24
- Lombok

## 运行与配置要点
- 服务端口：`server.port=8080`，上下文路径：`/api`
- 数据源：MySQL `property_management`（Druid 连接池）
- 前端代理：`/api` -> `http://localhost:8080`

## 前端
- Vue 3 + Vite
- Vue Router / Pinia
- Element Plus + Icons
- Axios / ECharts / Day.js
- Sass
- unplugin-auto-import / unplugin-vue-components
