# 运行清单

## 初始化数据库
- 在 MySQL 中创建数据库 `property_management`（字符集 `utf8mb4`）
- 导入 `小区物业管理系统/sql/init.sql`

## 后端启动
- 进入 `小区物业管理系统/property-management-backend`
- 修改 `src/main/resources/application.yml` 中的 `spring.datasource.*`
- 运行 `mvn spring-boot:run`
- 后端地址：`http://localhost:8080/api`

## 前端启动
- 进入 `小区物业管理系统/property-management-frontend`
- 安装依赖：`npm install`
- 启动开发环境：`npm run dev`
- 前端地址：`http://localhost:3000`
