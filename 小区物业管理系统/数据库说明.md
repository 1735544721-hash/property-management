# 小区物业管理系统 - 数据库说明

## 数据库信息
- **数据库名**：property_management
- **字符集**：utf8mb4
- **排序规则**：utf8mb4_unicode_ci

---

## 表结构说明

### 1. sys_user（用户表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键ID（自增） |
| username | VARCHAR(50) | 用户名（唯一） |
| password | VARCHAR(255) | 密码（BCrypt加密） |
| real_name | VARCHAR(50) | 真实姓名 |
| phone | VARCHAR(20) | 手机号 |
| email | VARCHAR(100) | 邮箱 |
| avatar | VARCHAR(255) | 头像URL |
| role | VARCHAR(20) | 角色：admin/staff/resident |
| status | TINYINT | 状态：0禁用/1启用 |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |
| deleted | TINYINT | 逻辑删除：0否/1是 |

---

### 2. building（楼栋表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键ID（自增） |
| building_name | VARCHAR(50) | 楼栋名称 |
| floors | INT | 楼层数 |
| units | INT | 单元数 |
| description | VARCHAR(255) | 描述 |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |
| deleted | TINYINT | 逻辑删除 |

---

### 3. house（房屋表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键ID（自增） |
| building_id | BIGINT | 楼栋ID（外键） |
| unit_number | INT | 单元号 |
| floor_number | INT | 楼层号 |
| room_number | VARCHAR(20) | 房间号 |
| area | DECIMAL(10,2) | 面积（㎡） |
| owner_id | BIGINT | 业主ID（外键） |
| status | TINYINT | 状态：0空置/1已入住 |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |
| deleted | TINYINT | 逻辑删除 |

**索引**：building_id, owner_id

---

### 4. repair（报修表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键ID（自增） |
| user_id | BIGINT | 报修人ID（外键） |
| house_id | BIGINT | 房屋ID（外键） |
| title | VARCHAR(100) | 报修标题 |
| content | TEXT | 报修内容 |
| images | VARCHAR(500) | 图片URL（逗号分隔） |
| status | TINYINT | 状态：0待处理/1处理中/2已完成 |
| handler_id | BIGINT | 处理人ID |
| handle_result | TEXT | 处理结果 |
| create_time | DATETIME | 创建时间 |
| handle_time | DATETIME | 处理时间 |
| update_time | DATETIME | 更新时间 |
| deleted | TINYINT | 逻辑删除 |

**索引**：user_id, status

---

### 5. fee（费用表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键ID（自增） |
| house_id | BIGINT | 房屋ID（外键） |
| fee_type | VARCHAR(20) | 费用类型：物业费/水费/电费/停车费 |
| amount | DECIMAL(10,2) | 金额 |
| fee_month | VARCHAR(10) | 费用月份（如2025-01） |
| status | TINYINT | 状态：0未缴/1已缴 |
| pay_time | DATETIME | 缴费时间 |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |
| deleted | TINYINT | 逻辑删除 |

**索引**：house_id, fee_month, status

---

### 6. notice（公告表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键ID（自增） |
| title | VARCHAR(100) | 公告标题 |
| content | TEXT | 公告内容 |
| publisher_id | BIGINT | 发布人ID |
| status | TINYINT | 状态：0草稿/1已发布 |
| create_time | DATETIME | 创建时间 |
| publish_time | DATETIME | 发布时间 |
| update_time | DATETIME | 更新时间 |
| deleted | TINYINT | 逻辑删除 |

**索引**：status

---

### 7. complaint（投诉建议表）

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键ID（自增） |
| user_id | BIGINT | 投诉人ID（外键） |
| type | VARCHAR(20) | 类型：投诉/建议 |
| title | VARCHAR(100) | 标题 |
| content | TEXT | 内容 |
| status | TINYINT | 状态：0待处理/1已处理 |
| reply | TEXT | 回复内容 |
| handler_id | BIGINT | 处理人ID |
| create_time | DATETIME | 创建时间 |
| handle_time | DATETIME | 处理时间 |
| update_time | DATETIME | 更新时间 |
| deleted | TINYINT | 逻辑删除 |

**索引**：user_id, status

---

## ER关系图

```
sys_user (用户)
    │
    ├── 1:N ──→ house (房屋) ←── N:1 ── building (楼栋)
    │              │
    ├── 1:N ──→ repair (报修)
    │
    ├── 1:N ──→ fee (费用) ←── N:1 ── house (房屋)
    │
    ├── 1:N ──→ notice (公告)
    │
    └── 1:N ──→ complaint (投诉建议)
```

## 初始数据

| 表 | 数据说明 |
|----|----------|
| sys_user | 1个管理员 + 1个物业员工 + 2个业主 |
| building | 4栋楼（2栋高层 + 2栋多层） |
| house | 6套房屋（2套已入住） |
| notice | 2条公告 |
| repair | 2条报修记录 |
| fee | 8条费用记录 |
| complaint | 2条投诉建议 |
