# 小区物业管理系统 - 部署说明

## 环境要求
- JDK 17+
- Node.js 18+
- MySQL 8.0+
- Maven 3.6+

---

## 启动步骤

### 1. 初始化数据库
```bash
mysql -u root -p < sql/init.sql
```

### 2. 修改数据库配置
编辑 `property-management-backend/src/main/resources/application.yml`：
```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/property_management
    username: root      # 改为你的用户名
    password: root      # 改为你的密码
```

### 3. 启动后端
```bash
cd property-management-backend
mvn spring-boot:run
```
后端地址：http://localhost:8080/api

### 4. 启动前端
```bash
cd property-management-frontend
npm install
npm run dev
```
前端地址：http://localhost:3000
